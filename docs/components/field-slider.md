## Properties
Note: For each property, the **bolded** value indicates the default setting.

- **Label** - The label text  
  Values: **string**

- **Helper** - Helper text  
  Values: **string**

- **Value** - Current slider value  
  Values: **0** (number)

- **Min** - Minimum value  
  Values: **0** (number)

- **Max** - Maximum value  
  Values: **100** (number)

- **Step** - Step increment  
  Values: **1** (number)
