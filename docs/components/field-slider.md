# Field - Slider

## Overview
The slider field combines the slider control with a label and helper text, making it easy to explain what the slider does and how to use it. Itâ€™s a great fit for forms, settings, and anywhere you want to give users a little extra guidance.

## Properties
Note: For each property, the **bolded** value indicates the default setting.

- **Label** - The label text  
  Values: **string**

- **Helper** - Helper text  
  Values: **string**

- **Value** - Current slider value  
  Values: **0** (number)

- **Min** - Minimum value  
  Values: **0** (number)

- **Max** - Maximum value  
  Values: **100** (number)

- **Step** - Step increment  
  Values: **1** (number)
