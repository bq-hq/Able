# Ctlr - Slider

## Overview
The slider control is the smooth operator of input fields, letting users pick a value (or a range) by sliding a handle along a track. Itâ€™s perfect for settings, filters, or anywhere you want to make number selection a little more fun and interactive.

## Properties
Note: For each property, the **bolded** value indicates the default setting.

- **Value** - Current slider value  
  Values: **0** (number)

- **Min** - Minimum value  
  Values: **0** (number)

- **Max** - Maximum value  
  Values: **100** (number)

- **Step** - Step increment  
  Values: **1** (number)

- **Disabled** - Whether the slider is disabled  
  Values: **false**, true
