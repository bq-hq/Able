# Progress

## Overview
The progress bar is the visual drumroll for your app, showing users how far along they are in a process. Whether itâ€™s uploading, loading, or completing a task, the progress bar keeps everyone in the loop with a clear, animated indicator.

## Properties
Note: For each property, the **bolded** value indicates the default setting.

- **Value** - Current progress  
  Values: **0** (number, percent)

- **Max** - Maximum progress  
  Values: **100** (number, percent)

- **Color** - Progress bar color  
  Values: **var** (any valid color variable)

- **Striped** - Striped animation  
  Values: **false**, true

- **Animated** - Animated progress  
  Values: **false**, true
